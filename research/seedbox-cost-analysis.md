# SeedBox Cost Analysis & Pricing Strategy
> Compiled: 2026-06-13

## COGS Breakdown Per Box

| Cost Item | Per Box |
|---|---|
| 5 seed packets (wholesale ~$0.65 ea via Bentley Seeds) | $3.25 |
| Fertilizer sample sachet | $0.60 |
| Padded bubble mailer (#0, bulk 500+) | $0.09 |
| Custom branding sticker (1K run) | $0.12 |
| Insert/info card (printed, bulk) | $0.10 |
| USPS First-Class postage (2 oz flat) | $1.90 |
| Packing labor (self-fulfillment, 5 min @ $15/hr) | $1.25 |
| Payment processing (Stripe ~2.9% + $0.30) | $0.88 |
| Platform/software fee (amortized) | $0.40 |
| **Total Loaded COGS** | **$8.59** |
| **Hard COGS (no labor/processing)** | **$6.06** |

## First Box Free — Break-Even Analysis

| CAC Scenario | Total Initial Outlay | Months to Break Even |
|---|---|---|
| $0 (organic) | $8.59 | 1 paid month |
| $10 (social ads) | $18.59 | 2 months |
| $20 (Meta ads) | $28.59 | 3-4 months |

Average gardening subscription retention: 4-8 months. First box free is viable.

## Key Suppliers
- Seeds: Bentley Seeds (bulk wholesale, $0.50-0.85/packet at 250-500 minimums)
- Mailers: JiaroPack or Amazon bulk (#0 poly bubble, $0.05-0.09/unit)
- Stickers: PrintRunner (1K run, $0.10-0.15/sticker)
- Shipping: USPS First-Class Large Envelope ($1.90 at 2oz, 2026 rates)
